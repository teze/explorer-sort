## [1.0.0] - 2026-05-23

### 新增功能
- ✨ 在资源管理器工具栏添加排序图标按钮
- ✨ 支持 6 种排序方式：
  - 默认排序（VS Code 原生）
  - 按名称排序
  - 按类型排序
  - 按大小排序（有限支持）
  - 按修改时间排序
  - 按创建时间排序（有限支持）
- ✨ 快速选择菜单，显示当前激活的排序方式
- ✨ 排序状态持久化，重启后保持
- ✨ 多工作区支持，每个工作区独立设置
- ✨ 命令面板集成，支持快捷命令
- ✨ 错误处理和降级机制
- ✨ 中文界面和提示信息

### 技术特性
- 🚀 快速响应，不阻塞 UI
- 💾 工作区级别的配置持久化
- 🔧 完整的 TypeScript 类型支持
- 📝 详细的文档和使用指南
- 🐛 健壮的错误处理

### 已知限制
- ⚠️ 按大小排序：VS Code API 不支持，使用文件优先模式替代
- ⚠️ 按创建时间排序：VS Code API 不支持，使用修改时间替代
